## Persyaratan

1. <a href="https://nodejs.org/en/download/">Node Js</a>
2. Node_modules
3. ReactJs

## Cara menjalankan aplikasi ?

1. Buka direktori aplikasi di CMD atau Terminal
2. Siapkan browser untuk membuka aplikasi
3. ketik `yarn start`
